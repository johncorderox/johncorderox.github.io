Bubble

Bubble is a 3 day project with 3 team members aimed towards the E-commerce market. The project idea was simply for people to use our service and select a number of dry cleaning services and the closest dry cleaning agency would continue the process. We were able to use a number of API's like Twilio for texting updates and Google Maps for the admin section.
