Logistics

Logistics is a Rails Applications that is targeted for companies in need of a secure and easy Auction system. What this app does differently is that we essentially remove the need for a freight broker and include everything he does in the admin panel. Independent freight transporation can bid on contracts for delivery and they're able to see distance, weather, and current contract price. This was a 3-day project with my teammates from coding dojo.
You can view the repo here.
