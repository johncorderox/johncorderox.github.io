Beach House API


The Beach House API is the first (v1) Rails API in the BH series. The API features versioning, caching, CORS, and basic CRUD access. The API comes packaged with a React JS frontend that calls the backend to perform GET and POST requests. What is Beach House? - They are my favorite band and it was a fun and simple way to create my first ever API. They are with Sub Pop and you can find their music here

You can also view the repo here.



Currently the react JS frontend has a very basic UI that is primarily used for API calls only. GET and POST are the only allowed HTTP requests for now, but below is my to-do list before releasing to the public!



  1. Add basic CRUD to the API
  2. Add CRUD to the frontend react API calls
  3. Re-do the UI
  4. Release the API
