Avita Medical

Avita Medical is the first project as a developer at Sunnybyte I was able to lead and architect on my own. The website is built with a custom bootstrap and avita theme css with Craft CMS (based off PHP) written on the backend. I made everything from the content integration to each module made in the backend CMS. Each module was rendered using the frontend templating engine "Twig" as well as custom craft CMS plugins like redactor.

You can view the live site here as well as Sunnybyte, a web and mobile developement LLC.
