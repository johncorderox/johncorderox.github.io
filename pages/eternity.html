Eternity Tracking

Eternity Tracking is a bug tracking system written in pure PHP and MySQL. What makes this project so robust is that it is easily customizable by any developer. It can be used by any start up or any organization in need of a free tracker.
You can view the repo here.



Clone the project into your desired directory:
git clone https://github.com/johncorderox/Eternity-Tracking.git

Update the following to your hosts credentials.
public $servername = 'localhost';
public $username = 'root';
public $password = '';
public $database = 'tracking';

Run the database.sql file and make sure the admin insert query has ran.

Login with the default admin credentials in the database query and enjoy~.
