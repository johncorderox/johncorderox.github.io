Playtest v1

Playtest is a Ruby web application that incorporates Amazon S3 data dumps for secure and versioning of admin notes, playtesters, and playtest accounts. I had the privilge of playtesting for Squanch Games, sister company of Rick & Morty © and recreated the process of application, to testing and to discharge all in one app. Key features include an application system, playtest collaborating, and data dumps to AWS S3 via Admin Panel.
You can view the repo here.



The Admin panel was one of the most important aspects for this playtest suite. Everything needs to be tracked in order to protect using secure access. The devise gem is used for password encryption and user model design so the default values were used.


Almost all actions are tracked and dumped using sqlite and Amazon S3 using standard HTTP codes 2xx 3xx and 5xx for reference.



Each application can be viewed by any admin and seperate notes and discussions can be made for each application.
